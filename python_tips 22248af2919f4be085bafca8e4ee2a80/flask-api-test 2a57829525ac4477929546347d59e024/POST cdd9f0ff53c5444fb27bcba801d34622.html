POST

  • 接下來要介紹 POST method，要示範的是新增使用者。
  • user傳json格式的新使用者帳密給我，我幫他新增這個人到目前的資料中。
  • 使用的 code如下 
@app.route('/user', methods=['POST'])
def create_user():
    user = request.get_json()
    user_list.append(user)
    return {
        'message': 'user created'
    }
  • 可以看到，我們設了一個 route 叫做 /user ，給 user 送 POST 訊息進來。
  • user送進來後，我們用 flast 的一個 function 叫 request 裡面的 .get_json() method，來獲取user傳進來的json訊息。然後，我就把這個json給append回原本的user_list之中，並回傳給user一個message說 user created
  • 完整的 code 如下：
from flask import Flask, jsonify, request

app = Flask(__name__) 

user_list = [
    {
        'username': 'abc',
        'password': 'abc'
    },
    {
        'username': '123',
        'password': '123'
    }
]

@app.route('/')
def helloworld():
    return "hello world"

@app.route('/users', methods=['GET'])
def get_users():
    return jsonify(user_list)

@app.route('/user', methods=['POST'])
def create_user():
    user = request.get_json()
    user_list.append(user)
    return {
        'message': 'user created'
    }

if __name__ == "__main__":
    app.run()
